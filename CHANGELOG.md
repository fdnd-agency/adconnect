# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.0.0] - 2026-02-18

### Added

- Admin login functionality with Directus authentication.
- Initial admin dashboard layout with sidebar navigation.
- Prettier and ESLint configuration for consistent code formatting.
- Content Security Policy (CSP) settings for improved security.
- Atomic design component structure (atoms, molecules, organisms).
- Created design for the admin pages.

### Changed

- Split routes into (public) and admin route groups.
- Moved FAQ section to a dedicated page.
- Refactored components into smaller, reusable parts following atomic design principles.
